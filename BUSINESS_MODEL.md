## 수익 모델: 프리미엄(Freemium) + 제휴(Affiliate) 하이브리드

### 이전 모델의 문제

- ₩9,900/월 구독: E-9 월수입 260만 중 생활비 제외 150~210만 원을 송금. 월 1만 원 = 송금 수수료 2건분.
- 외국인 결제 수단 제약: Toss Payments 해외카드 빌링 제한.
- 유료 전환 동기 부족: "이 기능 때문에 돈 낸다"는 킬러가 없었음.

### 새 모델

| 층 | 가격 | 포함 기능 |
|---|---|---|
| **Free** | ₩0 | 비자 점수 계산, E-7-4 시뮬레이터, 급여 계산기, 송금 비교(3회/월), 환율 알림, D-Day 알림, AI 서류 가이드 |
| **Premium** | ₩4,900/월 | 무제한 송금 비교, AI 계약서 스캐너, 서류 자동제출, 행정사 매칭 우선권, 프리미엄 서류 가이드, 광고 제거 |
| **Annual** | ₩3,920/월 (₩47,040/년) | Premium 동일, 20% 할인 |

### ₩4,900 근거

- 송금 수수료 1건(5,000원)과 동일 수준 → 심리적 저항 최소화
- E-9 월수입 260만 대비 0.19% → 가계 부담 미미
- WTP 데이터: 외국인은 "신뢰+편의성" 담보 시 프리미엄 지불 의사 있음
- Revolut Standard($0) → Premium($9.99) 전환 구조 벤치마킹

| 서류 제출 | "자동완성 도구이며 제출 책임은 본인" | 면책 체크박스 (강제) |

### 신규 — 급여 계산기 면책 (3중 구조)

**법적 근거 요약:**
- 공인노무사법 제27조: '상담·지도'는 비(非)노무사도 가능 (독점 업무 아님)
- 변호사법 사실행위 법리: 기계적 계산은 법률사무 아님
- 약관규제법 제7조: 중과실 면책 무효 → 경과실만 면책 가능
- 기존 선례: 알바몬, 사람인, 노동OK 등 동일 서비스 적법 운영 중

**3중 면책 구조:**

```
[Layer 1 — 결과 상단 라벨]
"📊 예상 급여 (모의계산)"
→ "예상", "모의계산" 프레이밍으로 확정적 판단이 아님을 명시

[Layer 2 — 결과 하단 면책]
"본 계산은 2026년 최저임금(시간급 ₩10,320) 기준 참고용 모의계산이며,
법적 효력이 없습니다. 실제 급여는 근로계약 조건에 따라 다를 수 있습니다."
→ 다국어 필수 (유저 선택 언어)

[Layer 3 — 전문가 연결 CTA]
"정확한 확인이 필요하다면:"
→ [📞 1350 외국인 노동 상담] 버튼 (전화 연결)
→ [👤 공인노무사 찾기] 버튼 (외부 링크)
→ 기능적 분리: 계산기(도구)와 법적 자문(전문가)의 명확한 경계
```

**설계 제약 5원칙:**

1. **"Wage Calculator"로 명명.** "Verifier", "Checker", "검증" 단어 금지.
2. **앱이 판단하지 않는다.** "위반입니다/적법합니다" 절대 표시 금지. 최저임금 기준액과 유저 시급을 나란히 보여주고 유저가 비교.
3. **수집 최소화.** 시급/근무시간/야간여부 3개만 입력. 외국인등록번호, 비자유형, 사업장명 수집 금지.
4. **Free 티어에 포함.** 변호사법 제109조 유상성 요건 원천 회피.
5. **면책은 유저 언어로.** 한국어만 면책 → 약관규제법 제3조(설명의무) 위반 가능. 다국어 필수.

### 면책 체크박스 대상 (체크 후에만 실행 가능)

- [ ] AI 계약서 스캔 (`parse-contract-ocr`)
- [ ] 서류 PDF 생성 (`render-immigration-pdf`)
- [ ] 팩스 발송 (`send-immigration-fax`)
- [ ] 행정사 매칭 요청 (`admin_match_requests` INSERT)

급여 계산기는 면책 체크박스 불필요 (인라인 면책으로 충분, 사용자 마찰 최소화).

## 경쟁 벤치마크 요약

| 경쟁자 | 핵심 기능 | Settle 차별점 |
|---|---|---|
| 비자인(VISA-IN) | E-7-4 점수 계산, 행정사 연결, 커뮤니티 | Settle = 점수 + 서류 자동제출 + 송금까지 원스톱 |
| 하이코리아 | 비자 연장 전자민원 | UX 최악, 모바일 불가 → Settle이 가이드 제공 |
| E9Pay/SentBe/Hanpass | 송금 | 단일 기능 → Settle은 비자+송금+급여 통합 |
| 1345 콜센터 | 20개 언어 상담 | 음성만 → Settle은 시각적+다국어+셀프서비스 |

### Settle의 킬러 포지셔닝

**"비자인의 점수 계산 + 하이코리아의 서류 제출 + 송금앱의 비교 = 하나의 앱"**

외국인이 3개 앱을 번갈아 쓰는 대신 Settle 하나로 해결.
